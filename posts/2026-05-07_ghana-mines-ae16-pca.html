Ghana Mines: AE-16 PCA Experiment

Project C · Experiment Report — 2026-05-07

Project C
Ghana Mines
Alpha Earth
Project C — Alpha Earth 16-band PCA experiment for Ghana mine detection. Tests whether AE’s learned land-cover embeddings improve mine detection over the S2 NDVI baseline. Key hypothesis: AE failed for Rwanda buildings (r=-0.29) because buildings are structure-level features, but mines are land-cover-level (forest→bare soil) — exactly what AE encodes.
Author

Andrea Salem

Published

May 7, 2026

1 Context

This report documents the AE-16 PCA experiment — the second entry on the Ghana mines experiment ladder, testing whether Google’s Alpha Earth learned embeddings improve mine detection beyond the S2 NDVI baseline.

For background on the Ghana mine detection pipeline, label strategy, baseline results (AUC 0.861), and the six-workstream improvement plan, see the Stage 1 report.

1.1 Why test Alpha Earth for mines?

Alpha Earth (64 learned embeddings at 10m) catastrophically failed for Rwanda building detection:

Experiment AUC District r (2023)
S2 NDVI (4 bands) 0.941 0.750
AE-only (64 bands) 0.921 -0.289
S2+AE (76 bands) 0.931 -0.260

Root cause: AE encodes land-cover semantics at landscape scale, not structure-level features like individual buildings. The 11/30 Rwandan districts with zero building pixels in AE predictions showed the model was classifying entire landscapes, not detecting structures.

But mines are different. Galamsey mines are a land-cover-scale transformation: dense tropical forest (NDVI ~0.8) replaced by bare soil, mud ponds, and turbid water (NDVI ~0.2). This is exactly the kind of signal AE was trained to encode. Supporting evidence:

  • Arezki et al. (2025) used Alpha Earth for mine detection in Africa with positive results
  • Mine footprints (0.1–10 ha) are landscape features, not point structures like buildings
  • The spectral contrast (forest → bare soil) is among the strongest land-cover transitions possible

Hypothesis: AE-16 will match or exceed the S2 NDVI baseline for mine detection, reversing the Rwanda failure pattern.

2.2 Why PCA? Why 16 bands?

The dimensionality problem: 64 input channels means 64x more parameters in the first conv layer. unet_tiny (~6,800 params) lacks capacity to learn in 64-dimensional space — this is why AE-only needed threshold=0.05 in Rwanda (model couldn’t separate classes).

PCA addresses three issues at once:

  1. Dimensionality: 64 → 16 bands = 4x fewer first-layer parameters
  2. Redundancy: AE’s 64 bands are learned embeddings, many highly correlated. PCA extracts orthogonal components.
  3. GPU memory: 16 bands fits unet_small comfortably on Quest or M4 Pro

Why 16 specifically: Sebastian’s convention for AE experiments. Retains ~90% of variance in typical satellite embeddings. Testable: if top-4 PCA components capture >80% of variance, a 4-band PCA experiment could also be informative.

2.3 Band selection methodology

PCA band selection script: src/preprocessing/10_band_selection_ae.py

Modes: - mode=select: Rank original bands by loading on top PCA components, pick top-k originals (interpretable but suboptimal) - mode=transform: Write new GeoTIFFs with PCA-transformed bands (optimal but non-interpretable)

For this experiment: mode=transform — we want maximum information compression, not interpretability. The model trains on PCA-transformed 16-band inputs.

Fallback heuristics if PCA disappoints:

Method Cost When
Mutual information ranking ~1 hr CPU If PCA components lack discriminative power
Correlation pruning Minutes Pre-filter before PCA
L1 on first conv layer 1 training run Nuclear option — let the CNN decide

5 Discussion

To be written after results.

Key questions to address:

  1. Does AE reverse the Rwanda failure pattern? If AE-16 matches or beats S2 NDVI for mines, this confirms the hypothesis that AE encodes land-cover features better than structure features.
  2. What do the PCA components capture? Spatial maps of top components may reveal whether AE separates forest/bare-soil/water — the key mine signature.
  3. Is 16 bands enough? If cumulative variance at 16 components is <80%, consider AE-24 or full AE-64 with unet_medium.
  4. Does AE reduce false positives? The NDVI baseline may confuse agricultural clearing with mining. AE’s richer feature space might better separate these.
  5. Implications for the experiment ladder: If AE-16 improves, proceed to Exp #3 (S2+AE-16 fusion). If not, skip to Exp #1 (+SWIR1) and investigate MI-based band selection.
